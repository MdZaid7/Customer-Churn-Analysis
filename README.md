# Customer Churn Analysis Dashboard

## Overview

This project focuses on analyzing customer churn using SQL and Power BI. The dataset contains information for **6,418 customers** with details such as demographics, contract type, internet services, payment methods, and customer status.

The main objective of this project is to understand why customers leave, identify the factors affecting churn, and build an interactive dashboard that helps businesses make better retention decisions.

---

## Tools Used

- SQL Server
- Power BI
- DAX
- Power Query

## Dataset Information

- Total Customers: **6,418**
- Dataset Type: Telecom Customer Churn
- Records Analyzed: **6.4K+**
- Features: **32 Columns**

## Key Insights

- Customers with month-to-month contracts have the highest churn.
- Customers with shorter tenure are more likely to leave.
- Competitor-related reasons contribute to a large portion of churn.
- Churn behavior varies across different states and age groups.
- Service usage also impacts customer retention.
